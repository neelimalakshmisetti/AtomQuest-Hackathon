3. Architecture Diagram

The application follows a decoupled full-stack architecture utilizing modern web technologies with secure Role-Based Access Control (RBAC).

graph TD subgraph Client Tier Browser[Users: Employee / Manager / Admin] end subgraph Presentation Tier UI[React.js SPA] Styles[Tailwind CSS] State[Auth Context] UI --- Styles UI --- State end subgraph Application Tier API[Node.js + Express API] Auth[JWT Auth Middleware] RBAC[Role Validation] API --- Auth Auth --- RBAC end subgraph Data Tier DB[(MongoDB)] Models[Mongoose ODM] Models --- DB end Browser -- "HTTPS" --> UI UI -- "REST API (JSON) + Bearer Token" --> API API -- "NoSQL Queries" --> Models
